# Sorting Algorithms by Complexity

## Time Complexity

| Algorithm       | Worst (Big O)| Average (Theta)| Best (Omega)|
|-----------------|-------------|-------------|-------------|
| Selection sort  | O(n²)       | O(n²)       | O(n²)       |
| Insertion sort  | O(n²)       | O(n²)       | O(n)        |
| Merge sort      | O(n log n)  | O(n log n)  | O(n log n)  |
| Heap sort       | O(n log n)  | O(n log n)  | O(n log n)  |
| Quicksort       | O(n²)       | O(n log n)  | O(n log n)  |
| Counting sort   | O(n + k)    | O(n + k)    | O(n + k)    |
| Radix sort      | O(n * d)      | O(n * d)      | O(n * d)      |
| Bucket sort     | O(n²)       | O(n + k)    | O(n + k)    |

## Space Complexity

- note: space complexity doesn't vary based on case

| Algorithm       | Worst / Average / Best Case |
|-----------------|------------------|
| Selection sort  | O(1)             |
| Insertion sort  | O(1)             |
| Merge sort      | O(n)             |
| Heap sort       | O(1)             |
| Quicksort       | O(log n)         |
| Counting sort   | O(n + k)         |
| Radix sort      | O(n + k)         |
| Bucket sort     | O(n)             |
